# Webpack bundling demo
This is a simple webpack-enabled application to demonstrate the power of webpack

- Babel Loader
- Sass Loader With Dart Sass
- HTMLWebpackPlugin
- WebpackDevServer
- Asset Resource Loader
- Source Maps
- Caching

## Usage

Run Dev Server (Port 3000)

```
  npm run dev
```

## Build for production

```
npm run build
```
